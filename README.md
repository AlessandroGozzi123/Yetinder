Yetinder<br>
Menší webová aplikace vytvořená ve frameworku Symfony (PHP).<br><br>
Uživatel může nahrát vlastního "yetiho" do databáze a následně procházet náhodně vybranými yeti.<br>
Každý yeti má výchozí skóre **10** a může být hodnocen tlačítkem **+1** nebo **–1**.<br><br>
